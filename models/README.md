# Models Folder

Place downloaded checkpoints/model directories here (or mount/symlink), then update config YAMLs accordingly.

Suggested structure:
- `models/FLUX.1-Kontext-dev/`
- `models/dinov2_vitb14_pretrain.pth`
- `models/dinov3_vit7b16_pretrain_lvd1689m.pth`
- optional: `models/sam_vit_b_01ec64.pth`

You can use symlinks if your models are stored elsewhere.
